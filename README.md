# Trades AI Landing Page

A modern, high-converting landing page for a B2B SaaS tool targeting UK tradespeople and installers.

## Features

- **Hero Section** with pain-point hook and clear CTAs
- **Core Features** showcasing the "No-Bullshit System"
- **Value Proposition Table** comparing "Us vs. The Old Way"
- **Dark Theme** with safety orange and navy blue accents
- **Fully Responsive** design optimized for mobile devices

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Design Notes

- Dark theme (gray-900 background) for professional, rugged feel
- Safety orange (#FF6600) for CTAs and accents
- Navy blue (#1E3A5F) available in theme config
- British, upfront, authoritative tone
- Mobile-first responsive design
